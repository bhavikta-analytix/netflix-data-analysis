# Netflix Data Analysis

## Project Overview

This project explores Netflix's content library using Python, Pandas, and Matplotlib.

The analysis focuses on:

- Movies vs TV Shows distribution
- Top countries producing Netflix content
- Content release trends over time
- Content ratings analysis
- Top genres on Netflix

## Dataset

Netflix Titles Dataset

Total Records: 8807

## Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- VS Code

## Key Insights

- Movies dominate Netflix's catalog.
- United States contributes the highest number of titles.
- Netflix content experienced rapid growth after 2015.
- TV-MA is the most common content rating.
- International Movies and Dramas are the most popular genres.
